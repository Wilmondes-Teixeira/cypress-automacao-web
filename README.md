# Projeto mínimo de automação de testes web com Cypress

- Estrutura básica criada sem page objects ou comandos customizados.
- Para rodar os testes, instale as dependências e execute `npx cypress open` ou `npm test`.

Estrutura criada:
- `cypress/e2e/comandos-basicos.cy.js`: Exemplo de teste básico
- `cypress/support/e2e.js`: Arquivo de suporte padrão
